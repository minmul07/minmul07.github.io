# Q5
[문제 상황]
그레고리력의 정확한 윤년 규칙은 다음과 같다.4로 나누어 떨어지면서 100으로 떨어지지 않으면 윤년이다. 400으로 나누어 떨어지는 것도 윤년에 포함된다.
년도를 입력받아 그것이 윤년이면 True/Flase를 출력하는 프로그램을 작성하시오.


```python
data = int(input("윤년 계산기\n연도 입력: "))
if ((data % 4 == 0) and not (data % 100 == 0)):
  print(f"{data}년은 윤년입니다.")
else:
  print(f"{data}년은 윤년이 아닙니다.")
```

    윤년 계산기
    연도 입력: 2000
    2000년은 윤년이 아닙니다.
    

# Q6
흥덕마트에서는 사과와 배를 농장에서 받아 온 후상자에 담아 판매하려고 합니다. 1박스에 사과는 12개, 배는 7개씩을 각각 담을 수 있습니다. 금일 농장에서 들어온 사과와 배의 개수를 입력 받아서 필요한 각 과일의 박스 개수가 몇 개이고 담은 후 남은 과일의 개수가 몇 개인지 출력하는 프로그램을 작성하시오. 상자를 가득 채우지 못하면 판매하지 않는다.


```python
apple = int(input("사과 개수 입력: "))
pear = int(input("배 개수 입력: "))
print(f"사과 {apple // 12}박스, 남은 사과는 {apple % 12}개 입니다.")
print(f"배 {pear // 7}박스, 남은 사과는 {pear % 7}개 입니다.")
```

    사과 개수 입력: 316
    배 개수 입력: 218
    사과 26박스, 남은 사과는 4개 입니다.
    배 31박스, 남은 사과는 1개 입니다.
    

# Q7
흥덕제약에서는 다음과 같은 성분 비율로 100ml의 시럽을 제조하고 있습니다. 각 성분의 재고량을 입력 받아 아래 표의 비율로 약을 제조할 때 하루 생산량인 100병이 가능한지 계산하는 프로그램을 작성하시오.

```python
A = int(input("성분 A의 용량 입력: "))

if ((B >= 50 * 100) and (B >= 3 * 100) and (C >= 25 * 100) and (D >= 12 * 100) and (E >= 10 * 100)):

```
