1. 리스트 lst에 저장된 값 [11, 22, 33, 44, 55, 66, 77, 88, 99]에서 출력 결과와 같이 주어진 구간의 값을 출력하는 프로그램을 작성하시오.


```python
lst = [11, 22, 33, 44, 55, 66, 77, 88, 99]
print(lst[3:5])
```

    [44, 55]

```python
flower = ['rose', 'lily', 'cosmos', 'Tulip']

```

3. 현재 lst에는 [1, 2, 3, 4]의 값이 저장되어 있다. lst에 값을 추가하여 출력 결과와 같이 되도록 프로그램을 작성하시오.


```python
lst = [1, 2, 3, 4]
lst.append(5)
lst.append(6)
print(lst)
```

    [1, 2, 3, 4, 5, 6]
    

4. 리스트 lst=[‘o', 'x', 'o', 'o', 'Nan', 'x']가 저장되어 있다. 이 때 ’Nan'은 측정되지 않은 값이다. 이 값을 리스트에서 제거하여 결과처럼 출력되도록 프로그램을 작성하시오.


```python
lst = ['o', 'x', 'o', 'o', 'Nan', 'x']
del lst[4]
print(lst)
```

    ['o', 'x', 'o', 'o', 'x']
    

피드백: 'Nan' 값이 여러개 존재할 때도 이 구현은 효율적인 구현이 될 수 있을까요?

응답: remove 연산자와 반복문을 이용하여 수정했습니다




```python
lst = ['o', 'x', 'o', 'o', 'Nan', 'x', 'Nan']
while (lst.count("Nan") != 0):
    lst.remove("Nan")
print(lst)

```

5. 항공 리스트 airList=[['인천','뉴욕'], ['인천', '타켄슈트'], ['싱가포르', '코펜하겐'], ['도쿄', '홍콩']]가 다음과 같다. 그런데 최근 코로나 19로 인하여 일부 출항 노선이 변경되었다. 결과를 참고하여 프로그램을 작성해 보시오


```python
airList= [['인천','뉴욕'], ['인천', '타켄슈트'], ['싱가포르', '코펜하겐'], ['도쿄', '홍콩']]
airList[3][0] = '하네다'
print(airList)
```

    [['인천', '뉴욕'], ['인천', '타켄슈트'], ['싱가포르', '코펜하겐'], ['하네다', '홍콩']]
    

피드백: '하네다'-->'광저우' 로 값을 한 번에 수정하려면 어떻게 구현해야 할까요?



```python
airList = [['인천','뉴욕'], ['인천', '타켄슈트'], ['싱가포르', '코펜하겐'], ['도쿄', '홍콩']]
airList[3] = ['하네다', "광저우"]
print(airList)
```
