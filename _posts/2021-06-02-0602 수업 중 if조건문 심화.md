[{문제 상황] 

 가위, 바위, 보 게임을 만들어 봅시다. 가위바위보 게임은 다음과 같이 진행됩니다. 
 
 ex1)
입력(가위:s, 바위:r, 보:p): s
비겼다. 

ex2)
입력(가위:s, 바위:r, 보:p): r
이겼다. 

ex3)
입력(가위:s, 바위:r, 보:p): p
졌다.


```python
while True:
  ans1 = input("플레이어 1 / 입력(s / r / p): ")
  ans2 = input("플레이어 2 / 입력(s / r / p): ")
  if (ans1 == ans2):
    print("비겼다.")
  elif ((ans1 == "r" and ans2 == "s") or (ans1 == "s" and ans2 == "p") or (ans1 == "p" and ans2 == "r")):
    print("플레이어1이(가) 이겼다.")
  else:
    print("플레이어2이(가) 졌다. ")
```

[문제 상황] 

 놀이 공원의 자유이용권 가격을 계산하는 프로그램을 작성하려고 합니다. 요금표는 아래와 같고, 고객의 나이와 입장 시간에 따라 금액이 달라집니다. 출력 예를 참고하여 구입요금을 계산하는 프로그램을 작성하시오. 단, 시간은 24시간제를 적용하므로 오후 5시는 17시로 표현한다.



```python
tm = int(input("현재 시간을 입력하세요. (0~23 정수): "))

if tm < 17: #시간대 확인
  adfee = 34000
  kidfee = 25000
else:
  adfee = 10000
  kidfee = 10000
#만약 nightfee 변수를 이용한다면, 변수가 하나 더 추가되고, if문을 한번 더 겨쳐야한다고 생각됨

fee = 0
while True:
  ip = input("고객의 나이가 몇세입니까? (종료: 정수 외 입력) ")
  if ip.isdigit() == True:
    if 13 <= int(ip) < 64:
      fee = fee + adfee
    else:
      fee = fee + kidfee
  else:
    print(f"요금은 {fee}원 입니다.")
    break
```
